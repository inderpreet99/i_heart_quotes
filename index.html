i_heart_quotes

I <3 Quotes Ruby Client

Overview

This is a ruby client to get random fortunes from IHeartQuotes.com.<\p>

Authors

Roberto Decurnex (nex.development@gmail.com)

Installation

If you are using Bundler you can easily add the following line to your Gemfile:

gem 'i_heart_qupotes'

Or you can just install it as a ruby gem by running:

$ gem install i_heart_quotes

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/robertodecurnex/i_heart_quotes
